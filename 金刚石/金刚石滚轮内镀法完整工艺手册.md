## 一、工艺总览与基本原理

### 1.1 内镀法的定义

内镀法（Internal Plating Method）是制造高精度金刚石滚轮的首选工艺。其核心思路是：在精密的模具型腔内，让镍-金刚石复合镀层从钢制基体外表面向外定向生长，最终使滚轮的外轮廓与模具型腔的内轮廓完全吻合。

可以类比为"在外模里铸造"：模具型腔的内表面精度决定了滚轮的最终精度，镀层只是充当了传递精度的介质。

### 1.2 为什么内镀法精度高于外镀法

外镀法（直接在基体上电镀金刚石，不做外模约束）的最终轮廓取决于镀层厚度的均匀性，而电镀的厚度均匀性在复杂型面上很难控制。内镀法通过模具强制约束了镀层的最终外轮廓，无论镀层厚度在局部有多少微小波动，外轮廓都由模具唯一确定。

## 二、模具系统设计与制造

模具是内镀法的灵魂。模具的精度直接等于滚轮的精度上限。

### 2.1 模具材料选择

| 材料 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **304不锈钢** | 耐腐蚀、易加工、可重复使用 | 热膨胀系数较大 | 通用，最推荐起步使用 |
| **316L不锈钢** | 耐氯离子腐蚀更强 | 价格高于304约30% | 高要求镀液、长期使用 |
| **6061/7075铝合金** | 轻便、导热好、易加工 | 不耐长期镀液腐蚀，需硬质阳极氧化处理 | 试制、小批量 |
| **45#钢+镀铬** | 成本最低 | 镀铬层可能剥落污染镀液 | 不建议，风险高 |

**推荐方案：** 304不锈钢，内腔表面精磨后抛光至Ra≤0.2μm。

### 2.2 模具结构设计

内镀法模具的基本结构为**多瓣组合式**：

```
                    模具分型结构示意（轴向剖视图）
            ┌─────────────────────────────────────┐
            │         上压盖（锁紧用）              │
            ├──────┬──────────────────────┬────────┤
            │      │                      │        │
            │ 左   │    模具型腔（内壁     │  右    │
            │ 半   │    为滚轮外轮廓的     │  半    │
            │ 模   │    负形）             │  模    │
            │      │                      │        │
            ├──────┴──────────────────────┴────────┤
            │         下底板（定位+密封）            │
            └─────────────────────────────────────┘
            
            型腔内部：
            ┌──────────────────────────────────┐
            │  ←模具内壁（Ra≤0.2μm）            │
            │      ░░░░金刚石颗粒░░░░░          │
            │         ← 间隙(埋砂层) →          │
            │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓    │
            │  ▓▓▓▓ 钢制基体(阴极) ▓▓▓▓▓▓▓▓    │
            │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓    │
            └──────────────────────────────────┘
            
            镀层从基体向外生长，包裹金刚石，填满间隙
```

**关键设计参数：**

| 参数 | 数值 | 说明 |
|------|------|------|
| 模具型腔与基体间隙(单边) | 金刚石粒径的1.5-2.0倍 | 例如120/140目(≈110μm)金刚石，单边间隙取165-220μm |
| 分型面数量 | 2瓣（轴向对开）或3瓣 | 直径<150mm用2瓣，>150mm用3瓣 |
| 分型面定位 | φ6-φ8定位销×4 | 保证合模精度<0.01mm |
| 拔模斜度 | 不设（直壁） | 型腔内部不设拔模斜度，靠分型脱模 |
| 模具壁厚 | 15-25mm | 保证结构刚度，防止电镀过程中变形 |
| 锁紧螺栓 | M10-M12，4-6颗 | 均匀分布，扭矩30-50N·m |

### 2.3 模具加工流程

```
    棒料/锻件毛坯
         ↓
    粗车外圆及端面（留量2-3mm）
         ↓
    钻孔/镗孔（加工内腔，留量0.5-1mm）
         ↓
    铣削分型面（配定位销孔）
         ↓
    热处理（去应力退火：450-500℃×4h，炉冷）
         ↓
    精车内腔（留磨量0.1-0.15mm）
         ↓
    数控磨/坐标磨内腔（至最终尺寸，公差±5μm）
         ↓
    手工抛光内腔（钻石研磨膏W1→W0.5→W0.25）
         ↓
    三坐标/轮廓仪检测内腔精度
         ↓
    清洗、涂脱模剂、待用
```

**内腔加工的核心精度要求：**

| 精度指标 | 要求 |
|---------|------|
| 型面轮廓度 | ≤0.005mm |
| 内腔圆柱度 | ≤0.003mm |
| 内腔表面粗糙度 Ra | ≤0.2μm（镜面级） |
| 上下端面平行度 | ≤0.01mm |
| 分型面平面度 | ≤0.005mm |

### 2.4 脱模剂涂覆

模具装配前，在内腔表面涂覆一层极薄的脱模剂，防止镀层与模具粘连。

**脱模剂配方（推荐）：**
- 方案A：硅油（二甲基硅油，粘度500cSt）+ 四氯化碳按1:20稀释，涂覆后晾干
- 方案B：硬脂酸锌粉末溶于丙酮，浓度2-3%，刷涂
- 方案C：市售电镀专用离型剂（如MacDermid或Atotech产品）

**涂覆方法：** 用脱脂棉蘸取脱模剂溶液，在模具内腔均匀擦拭一层，自然晾干10分钟后用干净棉布轻轻擦去表面多余物，只保留分子级薄层。涂覆后的模具应在24小时内使用。


## 三、基体加工

基体是承载镀层的钢制骨架，其精度决定了滚轮的安装精度和使用寿命。

### 3.1 基体材料

| 材料 | 适用场景 | 备注 |
|------|---------|------|
| **45#钢（调质处理）** | 通用型，大多数磨齿机滚轮 | 调质硬度HRC22-28 |
| **40Cr（调质）** | 要求较高强度和耐磨性 | 调质硬度HRC28-32 |
| **GCr15轴承钢** | 高端滚轮，要求高刚度 | 淬火+回火，HRC58-62 |

推荐起步使用45#钢调质处理，性价比最高。

### 3.2 基体加工流程与参数

```
    45#钢棒料下料（外径放量5-8mm，长度放量3-5mm）
         ↓
    调质处理：840℃淬火（水冷）+ 580-620℃回火（空冷）
        目标硬度：HRC22-28
         ↓
    粗车外圆（留量1.5-2mm）
    粗车端面（留量1mm）
         ↓
    钻镗内孔（安装孔，留量0.3-0.5mm）
         ↓
    去应力退火：400℃×3h，炉冷至200℃出炉
         ↓
    半精车外圆（留量0.3-0.4mm）
         ↓
    精磨外圆：
      砂轮：白刚玉WA80K，线速度35m/s
      工件转速：60-80rpm
      进给量：粗磨0.01-0.02mm/次，精磨0.003-0.005mm/次
      最终留量：外圆比模具型腔小（间隙值），即基体外径 = 模具型腔内径 - 2×(间隙)
         ↓
    精磨两端面（平行度≤0.005mm）
         ↓
    精镗/珩磨内孔（安装基准孔）：
      公差：H6级（例如φ40H6 = φ40⁺⁰·⁰¹⁶₀）
      圆柱度：≤0.003mm
      粗糙度：Ra≤0.8μm
         ↓
    去毛刺（各棱边倒钝R0.2-R0.5，不允许有翻边毛刺）
         ↓
    三坐标/内径千分表检测
         ↓
    防锈包装待用
```

### 3.3 基体待镀面的表面状态要求

| 指标 | 要求 |
|------|------|
| 待镀表面粗糙度 Ra | ≤0.8μm |
| 表面洁净度 | 无油污、无锈斑、无指纹、无氧化色 |
| 棱边倒角 | R0.3-R0.5圆弧过渡（避免尖端放电导致镀层烧焦） |
| 非镀面保护 | 若有不需要镀的区域，需在镀前做绝缘遮蔽 |


## 四、镀前处理

镀前处理是电镀成功的基础，90%的镀层结合力问题都出在前处理不彻底上。

### 4.1 处理流程

```
           ┌─────────────────────────┐
           │ 有机溶剂除油（可选）      │
           │ 丙酮或酒精擦拭           │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 化学除油                 │
           │ 配方见4.2，温度60-70℃    │
           │ 浸泡15-20min            │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 热水洗（50-60℃）        │
           │ 流动水冲洗1-2min         │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 冷水洗（室温）           │
           │ 流动水冲洗1-2min         │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 电化学除油（阴极除油）    │
           │ 见4.3                   │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 热水洗→冷水洗            │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 酸洗（活化）             │
           │ 见4.4                   │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 两道纯水洗（去离子水）   │
           │ 电导率<10μS/cm           │
           └────────────┬────────────┘
                        ↓
           ┌─────────────────────────┐
           │ 立即转入电镀槽           │
           │ （表面保持湿润，不可干燥）│
           └─────────────────────────┘
```

### 4.2 化学除油配方

| 组分 | 浓度(g/L) | 作用 |
|------|-----------|------|
| 氢氧化钠 NaOH | 20-30 | 皂化油脂 |
| 碳酸钠 Na₂CO₃ | 30-40 | 缓冲、乳化 |
| 磷酸三钠 Na₃PO₄·12H₂O | 30-40 | 软化水、分散污物 |
| 硅酸钠 Na₂SiO₃ | 3-5 | 乳化分散 |
| OP-10乳化剂 | 1-2mL/L | 表面活性剂 |

**工艺条件：** 温度60-70℃，浸泡时间15-20分钟。对于油污较重的基体，可辅以超声波（频率28-40kHz）。

**检测标准：** 水膜破裂试验——取出后用纯水冲洗，表面水膜应连续均匀覆盖至少30秒，无水珠聚缩，即为除油合格。

### 4.3 电化学除油（阴极除油）

| 参数 | 数值 |
|------|------|
| 溶液 | 与化学除油液相同，或专用电解除油液 |
| 温度 | 室温-50℃ |
| 电流密度（阴极） | 5-10 A/dm² |
| 时间 | 1-3分钟（基体为阴极，即工件接负极） |
| 注意 | 除油最后30秒可切换为阳极（反向），消除渗氢 |

**原理：** 工件作为阴极时，表面产生大量氢气气泡，机械剥离油膜。但长时间阴极处理会导致氢渗入基体，因此在最后半分钟切换极性（工件接正极，变为阳极），表面产生氧气泡，有助于驱除已渗入的氢。

### 4.4 酸洗（活化）

作用是去除基体表面的氧化膜，使金属晶格暴露，确保镀层与基体形成金属键结合。

**配方A（通用）：**

| 组分 | 浓度 |
|------|------|
| 盐酸 HCl（工业级36%） | 10%-15%（体积比） |
| 去离子水 | 余量 |

**工艺条件：** 室温（15-30℃），浸泡30-60秒。观察到基体表面颜色均匀变浅灰色、有轻微气泡生成即可，不可过度酸洗。

**配方B（精密件推荐）：**

| 组分 | 浓度(g/L) |
|------|-----------|
| 氨基磺酸 NH₂SO₃H | 50-80 |
| 氯化镍 NiCl₂·6H₂O | 10-20 |

**工艺条件：** 室温，浸泡30-90秒。此配方与后续镀液中的氨基磺酸镍体系兼容，即使有少量带入镀槽也不会造成严重污染。

**酸洗后处理：** 立即用纯水（电导率<10μS/cm）冲洗两道，表面始终保持水膜湿润状态，不允许干燥，在30秒内转入镀槽或带电入槽。

**注意：** 酸洗时间不可过长。过酸洗导致基体表面过度腐蚀、粗糙度增大，会严重影响镀层结合力和最终轮廓精度。


## 五、埋砂（上砂）工序

埋砂是将金刚石颗粒定量、均匀地填充到基体与模具之间环形间隙的操作。这是电镀前最后一道机械操作，金刚石分布的均匀性直接决定滚轮的使用寿命。

### 5.1 金刚石的准备

#### 5.1.1 金刚石选型

| 参数 | 磨齿机滚轮推荐值 | 说明 |
|------|----------------|------|
| 粒度 | 100/120目 或 120/140目 | 对应被修整砂轮粒度46#-80#（滚轮金刚石粒度应比砂轮粗一号） |
| 型号 | MBD₈ 或 SMD | 人造金刚石，MBD用于一般场合，SMD用于高强度要求 |
| 强度 | 静压强度≥20N（120/140目） | 确保在修整砂轮时不破碎 |
| 形状 | 等积形（接近球形） | 避免片状、针状颗粒，它们在镀层中结合力差 |

**金刚石品牌推荐（起步期）：**
- 河南黄河旋风（国产主流品牌，性价比高）
- 中南钻石（国产高端品牌）
- 郑州华晶（国产中高端）
- Element Six（进口，仅做高端订单参考）

#### 5.1.2 金刚石预处理（必须！）

金刚石在制造过程中表面会附着石墨、金属触媒和粉尘，必须预处理才能保证与镍镀层的结合力。

```
  金刚石原料
       ↓
  ┌───────────────────────┐
  │ 第一步：碱煮除油       │
  │ NaOH 15%溶液          │
  │ 煮沸 20-30 min       │
  │ 去离子水冲洗至中性    │
  └───────────┬───────────┘
              ↓
  ┌───────────────────────┐
  │ 第二步：酸洗活化       │
  │ HNO₃(浓) 30% + 水 70% │  注意：必须在通风橱内操作
  │ 煮沸 20-30 min       │  硝酸蒸气有强腐蚀性和毒性
  │ 去离子水冲洗至中性    │
  └───────────┬───────────┘
              ↓
  ┌───────────────────────┐
  │ 第三步：烘干           │
  │ 120℃ 烘干2h，彻底干燥 │
  └───────────┬───────────┘
              ↓
  ┌───────────────────────┐
  │ 第四步：筛分           │
  │ 用标准筛再过一遍       │
  │ 去除团聚和碎粒        │
  └───────────┬───────────┘
              ↓
  密封干燥保存，待用
```

### 5.2 埋砂操作

#### 5.2.1 装配顺序

```
  ┌──────────────────────────────────────────────┐
  │  1. 将下底板与下半模组装，定位销定位            │
  │  2. 将基体安装到下半模中心位置（用工装保证同心） │
  │  3. 检查环形间隙的均匀性（用塞尺沿圆周多点测量） │
  │     → 单边间隙偏差不大于±15μm                 │
  │  4. 向环形间隙中填入预处理过的金刚石颗粒         │
  │     → 用小勺或专用加砂器缓慢均匀填入             │
  │  5. 边填边用铜棒轻轻敲击模具外壁（振实）         │
  │     → 频率：每秒2-3次，力度适中                │
  │  6. 继续加砂至间隙完全填满、无法再压缩           │
  │  7. 盖上上半模和上压盖                          │
  │  8. 均匀拧紧锁紧螺栓（对称交叉拧紧，分2-3次）    │
  │  9. 再次检查同心度                              │
  └──────────────────────────────────────────────┘
```

#### 5.2.2 关键控制点

| 控制点 | 标准 | 检测方法 |
|--------|------|---------|
| 基体与模具同心度 | 偏差≤0.02mm | 塞尺沿4个方向测量间隙 |
| 金刚石填充密度 | 振实后体积不再减少 | 加砂至溢满后再振，如砂面下降继续补充 |
| 模具结合面密封 | 无可见缝隙 | 合模后用0.02mm塞尺检查，不应插入 |
| 锁紧力矩 | 30-50N·m（均匀） | 扭矩扳手 |

### 5.3 埋砂质量的判定

埋砂不足（间隙内有空隙）会导致：
- 镀层在该区域没有金刚石，形成"空白区"
- 空白区在使用中快速磨损，滚轮寿命大幅降低

埋砂过紧（金刚石颗粒被过度压碎）会导致：
- 金刚石碎裂，切削能力下降
- 碎粒阻碍镀液流通，局部镀不上

**理想状态：** 金刚石颗粒间紧密接触（点接触为主），但未发生显著的破碎。间隙内充满了金刚石颗粒，每个颗粒都与其邻近颗粒形成稳定的三点/多点支撑。


## 六、电镀工序（核心）

### 6.1 镀液配方（氨基磺酸镍体系）

氨基磺酸镍体系是制造金刚石滚轮的标准选型，因为其镀层内应力低、延展性好、能沉积较厚的镀层而不开裂。

| 组分 | 浓度范围 | 最佳值 | 作用 |
|------|---------|--------|------|
| 氨基磺酸镍 Ni(NH₂SO₃)₂·4H₂O | 300-450 g/L | **400 g/L** | 主盐，提供镍离子 |
| 氯化镍 NiCl₂·6H₂O | 5-15 g/L | **10 g/L** | 活化阳极，改善导电性 |
| 硼酸 H₃BO₃ | 30-40 g/L | **35 g/L** | pH缓冲剂 |
| 十二烷基硫酸钠 C₁₂H₂₅SO₄Na | 0.05-0.2 g/L | **0.1 g/L** | 润湿剂，防针孔 |
| 糖精 C₇H₅NO₃S | 0.5-2.0 g/L | **1.0 g/L** | 初级光亮剂，降低内应力 |
| 1,4-丁炔二醇 C₄H₆O₂ | 0.1-0.3 g/L | **0.2 g/L** | 次级光亮剂 |

**配液用水要求：** 去离子水，电导率≤5μS/cm。

**配液顺序：**
1. 在60-70℃的去离子水中溶解硼酸（硼酸难溶，需充分搅拌）
2. 加入氨基磺酸镍溶液（或固体溶于温水后加入）
3. 加入氯化镍
4. 十二烷基硫酸钠用少量温水溶解后加入
5. 糖精和丁炔二醇分别用少量温水溶解后加入
6. 补充去离子水至目标体积
7. 调整pH至4.0（用氨基磺酸或碳酸镍调节）
8. 小电流（0.2-0.3A/dm²）电解处理4-8小时，去除金属杂质
9. 过滤后使用（滤芯精度1-5μm）

### 6.2 工艺参数

| 参数 | 范围 | 标准值 | 控制精度 |
|------|------|--------|---------|
| 镀液温度 | 45-55℃ | **50±2℃** | ±1℃ |
| pH值 | 3.8-4.5 | **4.0±0.2** | ±0.1 |
| 阴极电流密度 | 0.5-2.0 A/dm² | **1.0 A/dm²**（主镀期） | 恒流±5% |
| 阳极材料 | 电解镍板（Ni≥99.9%） | 含硫镍饼更佳 | |
| 阳极与阴极面积比 | 1.5:1 至 2:1 | **2:1** | |
| 搅拌方式 | 机械或空气搅拌+阴极移动 | 空气搅拌（需除油过滤） | |

### 6.3 电镀过程分段控制

内镀法滚轮的电镀不是恒流一次性完成的，而是分阶段控制：

```
  ┌──────────────────────────────────────────────────────┐
  │ 阶段1：预镀（冲击镀）         时长：30-60 min        │
  │ ─────────────────────────────────────────────────    │
  │ 目的：在基体表面快速形成一层致密的纯镍底层，          │
  │      为后续金刚石复合镀提供牢固的锚定层               │
  │ 电流密度：0.3-0.5 A/dm²（低电流确保覆盖完全）         │
  │ 阳极面积：正常配置                                   │
  │ 预期厚度：约5-10μm                                   │
  │ 注意：此阶段镀液中不含金刚石（埋砂在模具间隙中，      │
  │      镀液无法大量流通，但间隙中的镀液会先沉积纯镍）   │
  └───────────────────────────┬──────────────────────────┘
                              ↓
  ┌──────────────────────────────────────────────────────┐
  │ 阶段2：慢速填充期             时长：2-4 h            │
  │ ─────────────────────────────────────────────────    │
  │ 目的：镀层缓慢从基体向外生长，逐步接触金刚石颗粒      │
  │      并将颗粒根部包裹在镍层中                          │
  │ 电流密度：0.5-0.8 A/dm²                              │
  │ 此阶段镀层开始接触到金刚石颗粒的底部                   │
  │ 金刚石颗粒尚未被完全包裹，与基体的连接尚不稳定         │
  └───────────────────────────┬──────────────────────────┘
                              ↓
  ┌──────────────────────────────────────────────────────┐
  │ 阶段3：主镀期（快速沉积）      时长：6-16 h           │
  │ ─────────────────────────────────────────────────    │
  │ 目的：以较高的沉积速率将金刚石完全包裹在镍层中        │
  │      镀层填满间隙的所有空隙                            │
  │ 电流密度：0.8-1.5 A/dm²                              │
  │ 镍沉积速率：约15-25 μm/h（1.0A/dm²时约18μm/h）        │
  │ 此阶段是埋入金刚石颗粒深度的决定阶段                   │
  │ 金刚石被埋入镀层约50%-70%的深度（最佳为60%）           │
  └───────────────────────────┬──────────────────────────┘
                              ↓
  ┌──────────────────────────────────────────────────────┐
  │ 阶段4：平整收尾期             时长：1-3 h            │
  │ ─────────────────────────────────────────────────    │
  │ 目的：在金刚石层外生长一层薄纯镍覆盖层                │
  │      使镀层表面与模具内壁完全贴合                      │
  │ 电流密度：逐渐降低至0.3-0.5 A/dm²                     │
  │ 总镀层厚度略超间隙值5-15μm，确保完全贴合              │
  │ 最终断电、取出                                          │
  └──────────────────────────────────────────────────────┘
```

### 6.4 镀液维护

镀液是活的系统，必须持续维护：

| 维护项 | 频率 | 方法 |
|--------|------|------|
| pH检测与调整 | 每班次 | pH计（精度0.01），用氨基磺酸降低或用碳酸镍升高 |
| 温度监控 | 连续 | 温控器+加热管，波动控制在±1℃ |
| 镍离子浓度 | 每周1-2次 | EDTA滴定法，控制在标称值±10% |
| 硼酸浓度 | 每周1次 | 甘露醇法滴定 |
| 氯化物浓度 | 每两周1次 | 莫尔法（硝酸银滴定） |
| 表面张力 | 每两周1次 | 滴重法/毛细管法，目标35-45 dyn/cm |
| 镀液过滤 | 连续 | 滤芯精度1-5μm，流量≥槽容积×3/h |
| 活性炭处理 | 1-2月/次 | 2-5g/L活性炭，搅拌2h后过滤去除 |
| 赫尔槽试验 | 每月1次 | 检查镀液综合性能变化趋势 |

### 6.5 电镀过程中的异常与对策

| 异常现象 | 可能原因 | 对策 |
|---------|---------|------|
| 镀层发黑/烧焦 | 电流密度过大 | 降低电流，增加搅拌 |
| 镀层起皮/剥落 | 前处理不彻底，或镀液被有机物污染 | 检查前处理流程，活性炭处理镀液 |
| 镀层针孔 | 润湿剂不足，或搅拌不足导致气泡滞留 | 补充十二烷基硫酸钠，增强搅拌 |
| 镀层粗糙 | 电流过大，或镀液中悬浮固体过多，或铁杂质超标 | 降低电流，加强过滤，小电流电解除铁 |
| 镀层应力大（弯曲开裂） | 糖精含量不足，或温度过低，或pH异常 | 补充糖精，调整温度/pH |
| 金刚石分布不均 | 埋砂不均，或模具与基体同心度偏差 | 改进埋砂操作，检查装配精度 |
| 镀层漏镀（局部未镀上） | 该区域绝缘遮蔽不良或基体表面污染 | 改善前处理，检查绝缘遮蔽 |


## 七、脱模工序

脱模是将镀完的滚轮从模具中取出的操作。这是整个流程中最容易损伤产品的环节。

### 7.1 脱模流程

```
  ┌───────────────────────────────────────┐
  │ 1. 电镀完成后，从镀槽取出模具组合体    │
  │ 2. 用纯水冲洗模具外表面，去除镀液残留  │
  │ 3. 整体放入温水（40-50℃）浸泡10分钟   │
  │    → 利用热膨胀差异松动模具           │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 4. 按对角线顺序松开锁紧螺栓（分2-3次） │
  │    → 严禁一次拧松一颗螺栓到底         │
  │    → 否则分型面偏载会挤伤镀层         │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 5. 轻轻取下上压盖                     │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 6. 轻微敲击模具外壁（铜锤或橡胶锤）    │
  │    → 沿圆周均匀敲击，由轻到重         │
  │    → 听到声音由闷变脆说明模具松动     │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 7. 使用脱模拉杆/千斤顶缓慢推开分型面   │
  │    → 或用专用拆模工具                  │
  │    → 严禁用螺丝刀等尖锐工具撬          │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 8. 半模逐一取下                        │
  │    → 如果某处粘连，用木槌轻敲该处外壁  │
  │    → 不可用力掰分型面                  │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 9. 取出滚轮，检查外观完整性            │
  │    → 检查是否有模具粘连撕裂痕迹        │
  │    → 检查金刚石镀层是否有局部剥落      │
  └───────────────────────────────────────┘
```

### 7.2 脱模困难的处理

如果脱模剂失效或模具表面有缺陷导致粘模：

1. **热胀冷缩法：** 将模具组合体加热到80-100℃（热水浸泡），然后快速冷却模具外壁（喷冷水），利用模具与镀层的热膨胀系数差异产生分离力。可重复2-3次。

2. **冷冻法（慎用）：** 将整体放入-18℃冰箱冷冻2小时，取出后在模具外壁局部加热（热风枪），利用温差分离。

3. **注意：** 绝对不能使用任何化学溶剂浸渗脱模，因为溶剂会渗透进入金刚石-镍镀层的微观孔隙，残留后影响使用性能。

4. **如果仍然脱不下：** 最坏情况下宁可破坏模具（线切割切开）也不可用蛮力损坏滚轮产品。一套模具成本300-3000元，但一个废掉的滚轮损失的是产品声誉和客户信任。


## 八、镀后处理

### 8.1 清洗

```
  脱模后的滚轮
      ↓
  超声波清洗（去离子水，40-50℃，10-15min）
      → 去除残留镀液和松动的金刚石碎粒
      ↓
  纯水冲洗（去离子水，电导率<5μS/cm）
      ↓
  压缩空气吹干（无油无水，压力0.3-0.5MPa）
      → 重点吹干金刚石颗粒间的缝隙
```

### 8.2 去氢处理（除氢脆）

电镀过程中基体会吸收一定量的氢，导致氢脆（材料变脆，使用中可能脆断）。去氢是提高滚轮可靠性的关键步骤。

| 参数 | 数值 |
|------|------|
| 温度 | 200-250℃ |
| 时间 | 2-4小时 |
| 升温速度 | 缓慢升温（2-3℃/min），避免热冲击 |
| 冷却方式 | 随炉冷却至<80℃后出炉 |
| 时机 | 电镀完成后8小时内必须进行，越早越好 |

**注意：** 温度不可超过300℃，超过此温度金刚石在空气中开始氧化失重，表面性能下降。

### 8.3 表面钝化（可选但推荐）

用钝化液在镀层表面形成一层极薄的保护膜，提高镍层的抗腐蚀能力。

**推荐配方：**

| 组分 | 浓度 |
|------|------|
| 重铬酸钾 K₂Cr₂O₇ | 5-10 g/L |
| 去离子水 | 余量 |

**工艺：** 室温浸泡30-60秒，纯水冲洗，压缩空气吹干。

**注意：** 含铬废液属于危险废物，必须收集后交有资质单位处理。


## 九、精修工序

精修是使滚轮达到最终交付状态的最后加工环节。用户之前的方案中确实遗漏了这个关键步骤。

### 9.1 精修的目的

刚从模具中取出的滚轮虽然外轮廓精度已经很高，但通常存在以下需要修整的问题：

1. 分型面处有飞边（镍层沿模具缝隙溢出形成薄片）
2. 端面有多余的镍沉积
3. 内孔可能有微量的镍沉积（即使做了遮蔽）
4. 个别金刚石颗粒出刃过高（超出镀层表面太多）
5. 安装基准面的精度需要复检和微调

### 9.2 精修流程

```
  ┌───────────────────────────────────────┐
  │ 9.2.1 去除飞边                         │
  │ ─────────────────────────────────────  │
  │ 方法：油石条（氧化铝或碳化硅，粒度      │
  │      180#-240#）手工打磨分型线           │
  │ 注意：只去除飞边薄片，不可损伤正常的    │
  │      金刚石镀层表面                      │
  │ 检测：手指沿分型线触摸，不应感到         │
  │      任何台阶或毛刺                      │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 9.2.2 端面修整                         │
  │ ─────────────────────────────────────  │
  │ 方法：在平面磨床上轻磨两端面            │
  │      砂轮：绿碳化硅 GC120K，粒度120#   │
  │      每次进给0.002-0.005mm             │
  │ 目的：去除端面多余镍层，保证平行度     │
  │ 注意：只磨端面，不磨外圆工作表面       │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 9.2.3 内孔修整                         │
  │ ─────────────────────────────────────  │
  │ 方法：内圆磨床或珩磨                    │
  │      去除内孔壁可能附着的镍层           │
  │ 保证安装基准孔精度：H6级               │
  │ 圆柱度≤0.003mm                         │
  │ 粗糙度Ra≤0.8μm                         │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 9.2.4 金刚石出刃修整（重点工序）        │
  │ ─────────────────────────────────────  │
  │ 目的：去除个别出刃过高的金刚石颗粒      │
  │      使镀层表面金刚石出刃高度一致       │
  │                                        │
  │ 方法A（推荐）：用细粒度油石条（WA400#） │
  │ 轻轻打磨镀层表面，力度极小               │
  │ → 只磨去过高的出刃，不磨损镍基体       │
  │                                        │
  │ 方法B（精密）：用小型砂轮机+金刚石      │
  │ 磨盘（粒度600#），低转速（<3000rpm），  │
  │ 轻触式修磨                              │
  │                                        │
  │ 检测：用工具显微镜在10-20倍下观察      │
  │      金刚石出刃高度应均匀一致           │
  │      无金刚石被完全打平或缺失的区域     │
  └───────────────────┬───────────────────┘
                      ↓
  ┌───────────────────────────────────────┐
  │ 9.2.5 外观终检与标识                    │
  │ ─────────────────────────────────────  │
  │ - 在非工作面上激光打标（型号/批次号/日期）│
  │ - 外观拍照留档                         │
  │ - 镀层表面应无裂纹、无起皮、无针孔     │
  │ - 金刚石分布均匀，无目视可见的空白区   │
  │ - 颜色均匀的银灰色（镍镀层正常颜色）   │
  └───────────────────────────────────────┘
```

### 9.3 精修注意事项

- 精修全程**不可使用任何油脂类冷却液**，金刚石镀层有微孔，吸油后无法彻底清除，使用时污染砂轮
- 修磨外圆工作表面时**力度必须极小**，金刚石硬度极高（莫氏10级）但脆性大，重压会导致颗粒碎裂
- 飞边去除后，分型线处允许有极细微的颜色差异（镍在此处沉积条件与别处略有不同），但绝不允许有台阶或毛刺
- 精修完成后立即用压缩空气吹净所有磨屑和粉尘


## 十、质量检验标准

### 10.1 出厂必检项目

| 检验项目 | 标准/公差 | 检测工具 | 检测频率 |
|---------|----------|---------|---------|
| 外轮廓尺寸 | 按图纸，公差±0.01mm | 三坐标/轮廓仪 | 100% |
| 轮廓度 | ≤0.005mm（IT4级）/ ≤0.01mm（IT5级） | 轮廓仪 | 100% |
| 圆柱度 | ≤0.005mm | 圆度仪/偏摆仪 | 100% |
| 径向跳动 | ≤0.005mm（相对安装孔） | 偏摆仪+千分表 | 100% |
| 端面跳动 | ≤0.01mm | 偏摆仪+千分表 | 100% |
| 内孔直径 | H6级 | 内径千分表/三坐标 | 100% |
| 镀层外观 | 无裂纹/起皮/针孔/漏镀 | 目视+10×放大镜 | 100% |
| 金刚石分布均匀性 | 无目视可见的空白区(>1mm²) | 10×-20×工具显微镜 | 100% |
| 金刚石出刃状态 | 出刃高度均匀，无整粒脱落 | 工具显微镜 | 抽样20% |

### 10.2 抽样破坏性检测（首件或首批次）

| 检验项目 | 标准 | 方法 |
|---------|------|------|
| 镀层结合力 | 无剥离 | 热震试验：加热至300℃保温30min后水淬（室温），反复3次 |
| 镀层厚度 | 按设计要求 | 金相显微镜（截面法） |
| 金刚石埋入深度 | 颗粒直径的50%-65% | 金相显微镜（截面法，放大200×测量） |
| 镀层硬度 | HV 350-450 | 显微维氏硬度计（载荷50g） |
| 金刚石浓度 | 按设计要求 | 金相图像分析（单位面积金刚石面积占比） |

### 10.3 检验环境要求

- 温度：20±2℃
- 相对湿度：≤65%
- 检验工具和滚轮需在恒温室中恒温1小时以上再测量
- 避免阳光直射和空调风口直吹


## 十一、常见缺陷与对策

| 缺陷 | 外观 | 根本原因 | 对策 |
|------|------|---------|------|
| 镀层起泡/剥落 | 鼓包或片状脱落 | 前处理除油不彻底；酸洗不够或过度 | 严格执行4.2-4.4流程，水膜试验必做 |
| 分型线台阶 | 分型处有明显高度差 | 合模时定位销松动或锁紧不均 | 检查定位销精度，均匀锁紧 |
| 金刚石空白区（漏砂） | 局部没有金刚石，只有镍 | 埋砂不实，该区域金刚石在电镀前已流失 | 改进埋砂振实操作，检查模具密封 |
| 金刚石脱落（掉粒） | 使用中金刚石整粒脱落 | 埋入深度不足（<40%），或金刚石预处理不好 | 延长主镀时间，严格执行金刚石酸煮预处理 |
| 镀层粗糙/结瘤 | 表面有粗糙的镍结节 | 电流密度过大；镀液中有悬浮物 | 降低电流，加强过滤 |
| 表面针孔 | 微小凹坑 | 润湿剂不足；搅拌不足导致氢气泡附着 | 补充润湿剂，改善溶液流动 |
| 轮廓超差 | 测量结果超出公差 | 模具加工精度不够；或电镀过程中模具变形 | 提升模具加工精度，增加模具壁厚 |
| 滚轮使用中异常磨损 | 寿命远低于正常值 | 金刚石粒度选择不当，或金刚石质量差 | 确认金刚石规格和品牌，做批次来料检测 |
