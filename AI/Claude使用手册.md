# Claude 使用手册

> 写给汶的私人指南。随手记，随时翻。

---

## 一、Claude 能做什么

Claude 不止是聊天。它能：

- **读写文件**：帮你整理笔记、写代码、改文档、批量重命名
- **上网搜索**：查资料、找教程、核实信息
- **执行命令**：在你的电脑上跑脚本、装依赖、处理数据
- **规划任务**：把复杂需求拆成步骤，一步步做完再汇报
- **生成图片/视频**：根据描述出图出视频（有对应插件时）

把它想象成一个**坐在你电脑旁边、能直接动手的帮手**，不只是出主意，是真的能干活的。

## 二、Skills：给 Claude 装"专业技能包"

### 什么是 Skill

Skill 是一份**专业指令文件**（`SKILL.md`），告诉 Claude 在某个特定场景下该怎么工作。比如"雅思阅读技能"会告诉它怎么帮你刷雅思题，"office-documents"让它知道怎么处理 Word/Excel/PDF。

装了对应 skill 之后，Claude 在那个领域的能力会明显提升。

### 怎么搜索 Skill

有两种找法：

**找社区已有的**

直接让 Claude 帮你搜：

> 帮我搜一下 GitHub 上有没有处理 Excel 表格的 Claude skill

或者自己用关键词搜：

> 搜一下 `site:clawhub.ai 数据分析`

**自己写 Skill**

如果你觉得某个重复性工作每次都要说一遍太麻烦，可以告诉 Claude：

> 帮我把这个工作流程写成 skill，以后每次遇到类似任务自动用

### 怎么安装 Skill

拿到 GitHub 仓库地址后，一句话：

> 安装这个 skill：[GitHub 地址]

Claude 会自动下载并加载。

你也可以直接口述需求，让 Claude 现场写一个。比如：

> 帮我创建一个 skill，功能是每次我让你查论文时，自动去 arXiv 搜索、提取摘要、翻译成中文

### 你当前已装的 Skill

目前你有这些技能可用：

| Skill | 用途 |
|-------|------|
| 雅思阅读技能 | 刷雅思阅读题、背单词、分析长难句 |
| office-documents | 读写 Word、Excel、PPT、PDF |
| quiet-musing | 深度思考框架（复杂问题自动激活） |
| image-gen-guide | 图片/视频生成指引 |

遇到对应场景时 Claude 会自动调用，不用你手动触发。

## 三、常用指令速查

### 文件操作

| 你想做什么 | 可以怎么说 |
|-----------|-----------|
| 读文件 | 「读一下 `笔记.md`」 |
| 写新文件 | 「帮我创建一个 `README.md`，内容是...」 |
| 改某段内容 | 「把第 12 行改成...」 |
| 在当前目录找文件 | 「找一下所有 `.py` 文件」 |
| 搜索文件内容 | 「在所有 md 文件里搜 \`关键词\`」 |
| 把文件发给我 | 「把这个文件发给我」 |

### 网络操作

| 你想做什么 | 可以怎么说 |
|-----------|-----------|
| 搜索信息 | 「搜索一下 Python 3.13 新特性」 |
| 读网页内容 | 「读一下这个链接 https://...」 |
| 打开浏览器操作 | 「帮我打开京东看 iPad 价格」 |

### 命令执行

Claude 可以直接在你电脑上跑命令（bash）。Windows 下它知道用啥 shell。

| 你想做什么 | 可以怎么说 |
|-----------|-----------|
| 跑 Python 脚本 | 「运行 `script.py`」 |
| 安装包 | 「帮我装一下 pandas」 |
| 查看 git 状态 | 「看看当前 git 状态」 |
| 批量操作文件 | 「把所有 `.png` 文件名里的空格换成下划线」 |

## 四、进阶技巧

### 任务分解

当任务比较复杂时，Claude 会自动帮你拆成小步骤，在左边列一个 todo 清单，做完一个勾一个。你也可以主动要求：

> 这个任务比较复杂，先列个计划再开始做

### 子代理（Subagent）

Claude 可以分身——把需要大量搜索、阅读的调研任务派给"另一个自己"在后台跑，主对话不受影响。适合：

- 需要大量搜索再汇总的调研
- 和当前任务并行、互不依赖的子问题
- 同时探索多个方向

你不需要刻意触发，Claude 会在合适时机自动使用。

### 后台任务与通知

有些操作（生成图片、视频、子代理等）是**非阻塞**的——提交后立刻返回，跑完了自动通知你。你不用守着等。

**如果在跑后台任务时你切换到其他窗口或暂时离开，回来时已经跑完的结果会自动送到你面前。**

### 定时任务

让 Claude 在特定时间自动做事：

> 每天早上 8 点帮我搜索昨晚的 AI 领域新闻并整理要点

> 下周三下午 3 点提醒我给导师发邮件

### 记忆

Claude 会记住你说过的重要信息——偏好、背景、常用设置。你不用每次都重复。它不会主动提起"我记得你xxx"，而是在需要时自然地用上这些背景。

如果你想让某条信息更牢固地保留：

> 记住：我的工作目录在 G 盘，C 盘空间紧张，以后安装东西优先选 G 盘

如果你想忘掉某条：

> 忘掉关于 xxx 的记忆

## 五、提一个好问题的技巧

Claude 越清楚你的**上下文**，回答越精准。几个小建议：

**给出具体场景**

❌ 「帮我写个爬虫」

✅ 「我想爬豆瓣读书 Top250 的书名和评分，用 Python，结果存成 CSV，放在桌面」

**说出你的真实需求，不只是操作**

❌ 「帮我找所有 PDF 文件」

✅ 「我在整理去年的财务票据，帮我找到所有 PDF 格式的发票并统计总额」

**遇到不满意，直接说哪里不对**

Claude 不会觉得被冒犯。你可以说：

- 「太啰嗦了，精简到三段」
- 「加个例子说明」
- 「这个方案在国内网络环境下跑不通，换一种思路」

**利用类比**

Claude 在"把抽象概念具象化"上很擅长。遇到难理解的东西：

> 用外卖配送的流程给我解释一下什么是供应链管理

## 六、你可能还不知道但很有用的

### 1. 让 Claude 帮你写提示词

你自己的提示词能力有限？反过来让它帮你写：

> 我要用 Midjourney 画一张「雨夜东京街头的赛博朋克便利店」，帮我写一段英文 prompt

### 2. 批量"翻译"格式

不只是语言翻译，格式转换也很强：

> 把这堆 JSON 数据转成 Markdown 表格

> 把这篇 Word 文档转成适合公众号排版的纯文本

### 3. 快速验证一个想法

不确定某个方案行不行？先问 Claude，省时间：

> 我想用 ffmpeg 把 200 个 mp4 文件每 5 秒截一帧，这个命令行方案有什么坑？

### 4. 用它管理它自己

你可以让 Claude 帮你：

- 整理 skill（「检查一下我现有的 skill，哪些重复了，哪些可以优化」）
- 优化工作流（「我最近经常让你做 A→B→C 三件事，能不能合并成一个命令」）
- 记录常用配置（「帮我把代理配置、API key 路径之类的信息整理成一个参考文件」）

### 5. 多人协作 / 频道

Claude 支持频道和私信，适合团队场景。你有需要的时候告诉它就行。

### 6. 设置个性化

你可以调整 Claude 的行为偏好：

> 搜索一下设置里有没有「默认语言」相关的选项

> 把回复风格调得更简洁一点
